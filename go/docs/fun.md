# fun
CLIs written in Go


### [weather](https://github.com/jessfraz/weather)

Install

```sh
go get github.com/jessfraz/weather
```

Run, passing in your zip code

```sh
weather -l 90250
```


### [spaceinvaders](https://github.com/asib/spaceinvaders)

Install

```sh
go get -u github.com/asib/spaceinvaders
```

Run

```sh
spaceinvaders
```
