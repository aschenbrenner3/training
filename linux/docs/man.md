# `man`
> **man** - format and display the on-line manual pages

![rtfm](images/rtfm.png)

### examples

```sh
man man
```

```sh
man ls
```

```sh
man whoami
```

### search
* Press `/`
* Enter search criteria
* Hit `Enter`

###### Next
Go to next match: `n`

###### Previous
Go to previous match: `Shift + n`
